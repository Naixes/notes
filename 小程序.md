# 微信小程序

## flex伸缩布局

常用属性

```css
.fa {
    /* 水平排列 */ 
    display:flex;
    /* 垂直排列 */
    flex-direction:column;
    /* 主轴居中 */
    justify-content:center;/* space-between */
    /* 交叉轴居中 */
    align-item:center;/* flex-end底部对齐 */
    flex-wrap:wrap;/* nowrap */
}
.item {
    flex:0 1 auto;
}
```
